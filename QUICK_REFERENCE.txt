## 🔧 EDIT BEFORE UPLOAD

**Change countdown (index.html):**
```javascript
let countdown = 5;  // Change to seconds you want
```

**Add image (gallery.html):**
Insert before `</div>`:
```html
<div class="gallery-item" data-src="images/new_gallery/ImageName.jpeg" onclick="openModal(this)">
    <img src="images/new_gallery/ImageName.jpeg" alt="Image">
</div>
```

**Change button color (index.html):**
Find `.enter-button` in `<style>` and modify:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

## 🆘 QUICK TROUBLESHOOT

| Problem | Solution |
|---------|----------|
| Images missing | Check file upload success, verify paths |
| No styling | Hard refresh: Ctrl+F5 or Cmd+Shift+R |
| Won't redirect | Ensure gallery.html exists |
| Zoom broken | Check JavaScript console (F12) |
| Slow load | Reduce image quality, enable GZIP |
